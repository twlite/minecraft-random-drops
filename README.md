# ecmacraft-template

This is a template for an ecmacraft plugin. You can use this as a starting point for your own plugin development.

## Getting Started

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Build the plugin (production):

   ```bash
   pnpm run build
   ```

3. Run the development server (development):
   ```bash
   pnpm run dev
   ```
